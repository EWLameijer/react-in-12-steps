React in 12 steps - learn and repeat the basics of React

Introduction

The internet is full of React tutorials, there is not much sense in making another one be it that I am looking for something that is

  1. Fairly up-to date
  2. Is more introductory, giving a basis for react instead of marching all the way to contexts and stores, which are not that useful in projects for beginners

One of the problems of tutorials is that people follow steps, and afterwards forget everything; similar to the practicals I did as a chemistry student. To make things stick, learning science describes "desirable difficulties." This tutorial aims to help the learning process in three ways:

  1. It clearly gives an overview/step by step plan, so you can link each action to its greater purpose.
  2. It provides a handy checklist of terms that you can study later with a system like Anki or my own FermiEn
  3. It tries to encourage you to do 'post-tutorial' work. More on that below.

What do I mean with post-tutorial work?

If you have followed this tutorial to a certain level (say level 3 or level 4), it helps if you try to reach the same level independently with an app of your own the next day. Like any professional developer, you are allowed to use Google. If you get stuck, you can always check your old code or this tutorial, but it also means that your knowledge is not yet solid enough and you should try again the next day or the earliest opportunity after that.

The steps!

  1. Create something that works - Let a tool create a react app for you
  2. Transform the template app into a very simple app of your own - Clean the demo app to make your own app basis
  3. Extend your app - Create your own components
  4. Use simple user input - Create simple input methods like buttons, learn to use state
  5. Use more complex user input - Learn to use controlled components for form input
  6. Create flexible displays: Learn to use ?: and map and keys to be flexible in the number of components
