Component: a sort of 'lego building block' you can use for websites; React allows you to write components, either class-based or function-based. Create-React-App: the old way to create applications in React; is not maintained well and should not be used anymore. Next.js: a framework that includes React and some other libraries that work well together React: a library that helps make web-apps that are fast and have relatively few bugs. state: the value of important variables, which can/should impact the display of an app. State is central to React. Vite: a tool to easily create Web apps in many frameworks (not only React) and loads them quickly
